 --- fast_rm について ----
                            2004/09/26 白水啓章

１．概要
　　高速なファイル＆ディレクトリ削除ツールです。
　　（…explorer の余りの遅さ＆問い合わせの多さに頭に来て、
　　　高速＆問い合わせなしの強制削除を目的に作成）

　　MAX_PATH(260文字) を越えた位置のファイルも削除できます。(UNICODE版)

２．動作環境
　　WinNT/2000/XP : fast_rm   (UNICODE APP)
　　Win95/98/Me   : fast_rm95 (ANSI APP)

３．使い方
　　fast_rm [-s(ilent)] target_dirs/files...

　　-s : 終了時以外の表示を抑制します。

４．注意（危険なところ）
　　オプション＆問い合わせなしに、指定ディレクトリを再帰的に削除します。
　　問い合わせなしに、read only file を（可能なら）削除します。
　　また、万一バグがあれば、悲惨な結果になる可能性があります。
　　（「５．免責」を熟読のこと）
